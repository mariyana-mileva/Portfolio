Who Am I?

Mariyana Mileva

I am Mariyana, a 21-year-old student from Sofia, Bulgaria, currently pursuing a degree in ICT and Media Design in the Netherlands. My academic journey has fueled my passion for front-end development and UI/UX design, and I am eager to apply my skills and knowledge in a professional setting.

Why ICT?

My journey into ICT began in the fifth grade when my mother enrolled me in one of the top software academies in the country after recognizing my keen interest in computers. I continued to attend these classes until the seventh grade, when I had to choose a high school. Still passionate about programming, I applied to a prestigious school specializing in foreign languages, mathematics, and informatics. After five enriching years there, I decided to further my studies in ICT at Fontys University of Applied Sciences.

Erion Kapaj's Portfolio

Pixel Playground

Our team, 'Mna4,' developed a knowledge database website for the Video Lab at our university to provide detailed information about lab technology and tutorials on proper usage. We conducted user research, created wireframes and prototypes using Figma, and developed the site with Vue.js. I contributed to the user interface design, conducted usability testing, and gathered user feedback through surveys and interviews. The project included multiple feedback sessions and a rigorous design and development process, resulting in a user-friendly resource for students, faculty, and staff to easily access and utilize lab equipment information.

View Project

Weather API

For a personal project, I developed a dynamic weather application using HTML, CSS, and JavaScript with a weather API. The goal was to learn external APIs, geolocation, and create an engaging interface with real-time weather animations. I structured the HTML for location display, weather info, and animated elements, and applied CSS for a consistent, appealing design. JavaScript integration fetched real-time weather data and geolocation determined the user's location to display relevant weather info. The UI dynamically updated to reflect weather conditions, like showing a rainy background for rain and a sunny background for clear skies.

View Project

Guess The Number Game

For a personal project, I created an interactive "Guess the Number" game using HTML, CSS, and JavaScript. The project aimed to teach object-oriented programming concepts through the use of classes and to develop a user-friendly UI playable directly in a web browser. I designed the HTML to include elements for the game title, instructions, guess input field, and interaction buttons. CSS was applied to enhance the visual presentation, ensuring a cohesive and aesthetically pleasing design. JavaScript was used to implement the game logic, including event listeners for capturing user inputs and updating the game state dynamically.
